# PIXEL HEALTH — Product Plan

PIXEL HEALTH is a gamified health tracking app that collects real-time data from Apple Watch (HealthKit) and Galaxy Watch (Health Connect), streams it through Apache Kafka, and displays it through a retro pixel-art RPG-style dashboard with AI-generated avatars.

> "Your Health, Pixelated"

This repo contains the interactive product plan and design specification, viewable at [pixel-health-dubois.github.io/figma-plan](https://pixel-health-dubois.github.io/figma-plan/).

## Roadmap

| Milestone | Focus | Status |
|-----------|-------|--------|
| MVP | Live heart rate display (HealthKit + Health Connect) | In Progress |
| M2 | User profiles, extended metrics, AI avatar generation | Planned |
| M3 | WHOOP integration, advanced HRV analytics | Planned |
| M4 | Social features & challenges | Planned |
| M5 | AI-powered insights & anomaly detection | Planned |
| M6 | Export & third-party integrations | Planned |
